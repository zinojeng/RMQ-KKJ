# 台中出發北九州行程比較

本資料夾用來整理「台中出發、日本北九州」旅行社套裝行程的研究紀錄與後續 Excel 比較需求。

## 網頁版

直接開啟 [index.html](index.html) 可使用互動比較頁。網頁支援推薦分類、無自由活動篩選、價格／分數排序，以及逐團展開五日行程；GitHub Pages 可直接以 repository root 發布。

## 最終確認範圍

- 出發機場：台中國際機場 / 清泉崗 `RMQ`
- 抵達機場：日本北九州機場 `KKJ`
- 航空公司：中華航空 / 華航 `CI`
- 去程航班：`CI7380`，台中 `RMQ` 15:20 -> 北九州 `KKJ` 18:45
- 回程航班：`CI7381`，北九州 `KKJ` 19:45 -> 台中 `RMQ` 21:40
- 適用期間：2026-09-01 至 2026-10-23
- 出發日：每週二、五、日

## 已排除的混淆項目

- 星宇航空 `STARLUX / JX`：台中飛九州的航點曾被誤認為本案條件，但本案最後確認不採用星宇航空。
- 星悅航空 `STARFLYER / 7G`：官網活動為桃園 `TPE` 往返北九州 `KKJ`，不是台中出發。
- 台中飛熊本 `KMJ` 商品：屬九州旅遊圈商品，但不是本案指定的台中直飛北九州機場商品。
