# 最終 Excel 規格

## 篩選規則

只納入同時符合以下條件的商品：

- 台中 `RMQ` 出發
- 北九州 `KKJ` 抵達
- 中華航空 `CI7380 / CI7381`
- 出發期間落在 2026-09-01 至 2026-10-23

若商品頁只寫「台中出發九州」或「九州包機」，但未明確列出華航台中北九州航班，應標示為「待確認」，不要直接列為符合。

## 建議工作表

### 1. 總覽比較

欄位：

- 旅行社
- 商品名稱
- 商品網址
- 出發日期
- 天數
- 價格
- 航班
- 抵達與回程時間
- 主要城市
- 住宿亮點
- 餐食亮點
- 自由活動
- 購物站
- 估計拉車負擔
- 推薦族群
- 綜合評分

### 2. 逐日行程

欄位：

- 旅行社
- 商品名稱
- Day
- 城市 / 區域
- 景點
- 早餐
- 午餐
- 晚餐
- 住宿
- 自由活動
- 購物 / 免稅店
- 當日拉車估算
- 備註

### 3. 住宿餐飲摘要

欄位：

- 旅行社
- 商品名稱
- 飯店名稱或候選飯店
- 飯店等級
- 是否溫泉住宿
- 是否連泊
- 早餐次數
- 午餐自理次數
- 晚餐自理次數
- 特別餐食
- 住宿風險

### 4. 推薦與判讀

欄位：

- 排名
- 推薦商品
- 推薦理由
- 最適合族群
- 最大優點
- 最大缺點
- 是否適合長輩
- 是否適合第一次去九州
- 是否適合重視自由活動者
- 是否適合重視飯店者

### 5. 資料來源

欄位：

- 旅行社
- 商品名稱
- 網址
- 擷取日期
- 航班是否明確
- 價格是否明確
- 飯店是否明確
- 待確認事項

## 判讀原則

- 航班正確性優先於行程吸引力。
- 同一聯營商品若由多家旅行社販售，應列出價格、贈品、付款條件與客服差異。
- 拉車時間可先用城市距離估算，但要標註不是旅行社保證時間。
- 若飯店只列候選名單，住宿等級應用區間或風險註記呈現。
- 推薦排名應分成「整體最佳」、「飯店品質最佳」、「自由度最高」、「長輩最適合」、「CP 值最高」。
