# 前次對話摘要

## 原始需求

使用者一開始想搜尋「最近台中有航空公司飛到北九州」後，哪些旅行社推出相關套裝行程。初步印象是只有華府旅行社，因此希望找出其他旅行社，並比較行程差異、優缺點與推薦方案。

## 初步整理過的旅行社與商品類型

前次對話曾提到以下旅行社或商品方向：

- 華府旅行社：九州三列車、熊本機加酒自由行等特色商品。
- 雄獅旅遊：北九州深度型行程，包含高千穗峽、由布院、阿蘇、九州自然動物園與溫泉住宿。
- 易遊網：南北九州全覽型行程，涵蓋熊本、鹿兒島、宮崎、福岡等地。
- 泰元旅行社：與易遊網相近的聯營商品，價格或住宿安排可能不同。
- 星浩旅遊：南北九州全覽型商品，住宿安排略有差異。

當時曾把商品分成三類：

| 類型 | 代表商品 | 優點 | 缺點 |
| --- | --- | --- | --- |
| 北九州深度型 | 雄獅 | 拉車較短、節奏較悠閒、適合長輩或第一次去九州 | 南九州景點較少 |
| 南北九州全覽 | 易遊網、泰元 | 景點最多、CP 值高、一次跑多個縣市 | 拉車較久、節奏較趕 |
| 特色鐵道團 | 華府三列車 | 鐵道體驗強、行程較有特色 | 價格較高、南九州較少 |

## 後續修正

前次對話中曾發生三個航空條件混淆：

1. 一開始把「台中出發九州」與「星宇航空台中飛熊本」混在一起。
2. 後來又看到 `STARFLYER` 官網，誤以為是星宇航空；實際上 `STARFLYER` 是日本星悅航空，不是台灣星宇航空。
3. 使用者最後提供華航包機資訊，確認真正要找的是「中華航空台中直飛北九州」秋季限定包機。

## 最終確認條件

最終 Excel 應完全依照以下條件重新整理：

- 航空公司：中華航空 / 華航
- 航班：`CI7380 / CI7381`
- 航線：台中 `RMQ` 往返北九州 `KKJ`
- 期間：2026-09-01 至 2026-10-23
- 出發日：每週二、五、日
- 商品類型：旅行社套裝行程、團體旅遊、機加酒自由行皆可納入，但必須確認使用上述華航台中北九州包機。
